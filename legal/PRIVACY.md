## 日本語

### 1. はじめに
「集中タイマー (focus-timer)」(以下「本拡張機能」) は、ADHD/ASD 特性を持つ方や、その保護者・支援者の集中作業を支援する Chrome 拡張機能です。本ポリシーは、本拡張機能の利用にあたっての個人情報の取り扱いについて説明します。

### 2. 収集する情報
本拡張機能は、いかなる個人情報も収集しません。具体的には以下の通りです。

- 氏名・メールアドレス等の個人識別情報を収集しません
- IP アドレス・端末情報・位置情報を収集しません
- 閲覧履歴・利用履歴を外部に送信しません
- アクセス解析ツール (Google Analytics 等) を一切使用しません
- 広告配信を行わず、第三者広告ネットワークと連携しません

### 3. ローカル保存データ
本拡張機能は、以下のデータを利用者の端末上 (`chrome.storage.local`) にのみ保存します。これらは外部サーバーへ送信されません。

- タイマー設定 (集中時間、休憩時間、サウンド等の設定値)
- セッション統計 (完了したポモドーロ回数、累計時間)
- Premium ライセンス状態 (購入済みフラグ、トライアル開始時刻)
- 言語・テーマ設定

これらのデータは Chrome 拡張機能の管理画面から本拡張機能を削除することで完全に消去されます。

### 4. 外部通信
本拡張機能は通常動作中、いかなる外部サーバーとも通信しません。例外として、Premium 機能の購入時のみ、Stripe Checkout (https://checkout.stripe.com) の決済ページを新しいタブで開きます。決済情報の取り扱いは Stripe のプライバシーポリシーに従います。

### 5. 子供のプライバシー
本拡張機能は不登校児・発達特性児を含む子供の利用を想定していますが、子供から個人情報を収集することは一切ありません。広告・外部リンク・SNS 連携も一切含まれません。

### 6. 権限について
本拡張機能が要求する権限は、機能要件を満たす最小限のものに限定されています。各権限の用途は manifest.json および本拡張機能のストア掲載ページに記載されています。

### 7. 改定
本ポリシーを変更する場合、本拡張機能のアップデート時にバージョン番号とともに告知します。

### 8. お問い合わせ
本ポリシーに関するお問い合わせは、本拡張機能の GitHub リポジトリの Issue よりお願いします。

## English

### 1. Introduction
"Focus Timer (focus-timer)" (the "Extension") is a Chrome extension designed to support focused work for individuals with ADHD/ASD traits and their caregivers or support professionals. This policy explains how personal information is handled when using the Extension.

### 2. Information We Collect
The Extension does not collect any personal information. Specifically:

- We do not collect personally identifiable information such as name or email address.
- We do not collect IP addresses, device information, or location data.
- We do not transmit browsing or usage history to external services.
- We do not use any analytics services (such as Google Analytics).
- We do not display ads and do not integrate with any third-party advertising networks.

### 3. Locally Stored Data
The Extension stores the following data only on the user's device (via `chrome.storage.local`). This data is never transmitted to any external server.

- Timer settings (focus duration, break duration, sound preferences, etc.)
- Session statistics (number of completed Pomodoros, cumulative time)
- Premium license state (purchase flag, trial start timestamp)
- Language and theme preferences

All such data is fully erased when the Extension is uninstalled from Chrome's extension management page.

### 4. External Communication
Under normal operation, the Extension does not communicate with any external server. The sole exception is when a user initiates a Premium purchase, in which case the Stripe Checkout page (https://checkout.stripe.com) is opened in a new tab. Payment data handling is governed by Stripe's own privacy policy.

### 5. Children's Privacy
The Extension is intended for use including by children with school-refusal or developmental traits, but no personal information is collected from children. The Extension contains no advertisements, external links, or social media integrations.

### 6. Permissions
The permissions requested by the Extension are strictly limited to the minimum required to provide its features. The purpose of each permission is documented in manifest.json and on the Extension's Chrome Web Store listing.

### 7. Changes
If this policy is changed, the change will be announced together with a version bump when the Extension is updated.

### 8. Contact
For inquiries about this policy, please open an issue on the Extension's GitHub repository.
