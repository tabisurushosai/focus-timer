# ストア掲載情報 - focus-timer

## 名前 (32字以内)
Focus Timer — ポモドーロタイマー

## 概要 (132字以内、必須)
ポモドーロタイマーで集中作業をサポート。25分作業 + 5分休憩のサイクルで生産性を最大化。通知音と統計機能付き。
Pomodoro timer to boost focus. 25min work + 5min break cycles with notifications and daily stats.

## 詳細説明 (本文)
集中力を高めるシンプルなポモドーロタイマー Chrome 拡張機能です。

【主な機能】
- 25分作業 + 5分休憩の標準ポモドーロサイクル
- カスタマイズ可能なタイマー時間 (作業/休憩の長さを変更可)
- 終了時の通知音 (音量調節可)
- 1日の完了サイクル数を記録
- 統計画面で週次・月次の集中時間を可視化
- 完全オフライン動作 (外部サーバーへのデータ送信なし)
- 日本語 / 英語 対応

【こんな人にオススメ】
- 集中力が続かない方
- リモートワークで作業のメリハリをつけたい方
- 学生・受験生の勉強リズム作りに
- ADHD などで時間管理が苦手な方

【プライバシー】
すべてのデータは端末内 (chrome.storage.local) に保存され、外部サーバーへ送信されることはありません。
広告なし・完全無料 (Premium 機能のみ任意で $3 買い切り)。
